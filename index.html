  • About

    This is a small personal design project for an ear training tool using solfège. It was conceived as an exercise in interface design as well as an introduction to newer HTML5/CSS developments.


    I attempted to accomplish this project with as little JavaScript as I could get away with. As a result, most of the effects you see are heavily implemented via CSS. My use of jQuery is largely limited to handling the tone generation and toggling classes.


    Since I am not primarily a front-end developer (if you couldn't tell already), I thought it would be a good opportunity to round out my skillset.


    Right now, it is still a work in progress. I am currently planning on:


    • Fixed jQuery event handling for clicking on the radial menu instead of hovering.
    • A game loop to continuously generate new tones ad infinitum without refreshing.
    • Configurable solfege pronunciations.
    • Configurable octaves and keys.
  • How to Use

    Simply press the icon in the center of the screen to play a tone and activate the radial menu. Hover over the name of the tone you think it is, and the ring will turn green if you are correct, red if not.


    NOTE: Currently, you will need to refresh the page in order to generate a new tone (I'm working on it).
